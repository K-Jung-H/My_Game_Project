문제점

재질을 어디 구조체에 바인딩 할 것인가?



방법 1

* 현재 Mesh 에서 사용할 재질 ID를 Sub Mesh 별로 저장 중 -> 재질을 별도로 저장하지 않고 사용하기 전에 접근하여 바인딩 호출하기



방법 2

* 셰이더의 PSO나, 인스턴싱을 고려하면 재질에서 사용할 셰이더를 관리하는 구조가 되어야 함 ->  재질을 매쉬와 같은 계층인 랜더러 컴포넌트에서 ID를 관리하고 바인딩 하는 구조
